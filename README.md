## ğŸš€ Getting Started

### Prerequisites

Ensure you have Python 3.7+ installed along with the following packages:

```bash
pip install pandas numpy matplotlib seaborn jupyter scikit-learn
```

### Installation

1. Clone this repository:
```bash
git clone https://github.com/sgr9/Data_Analysis.git
cd Data_Analysis
```

2. Navigate to the desired project folder:
```bash
cd Banking_Dashboard
# or
cd customer-trends-data-analysis
```

3. Launch Jupyter Notebook:
```bash
jupyter notebook
```

4. Open the relevant notebook file and run the cells to reproduce the analysis.
